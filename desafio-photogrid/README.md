# Desafio Photo Grid # 

Página estilo Photo Grid desenvolvido em HTML e CSS usando as propriedades do Flexbox e o conceito Mobile First. 

Demo: https://womakerscode.github.io/front-end-study-group-rj/demo-photogrid/
